# Claude Code에서 MCP 서버 3분 만에 연결하는 법

---

## 1. 요약 (3줄)
- Claude Code의 최신 업데이트에서 MCP(Model Context Protocol) 서버를 로컬에서 바로 붙일 수 있게 됐습니다
- 설정 파일 하나만 추가하면 3분 안에 연결 완료
- json 파일을 프로젝트 루트에 생성

## 2. 왜 중요한가
Claude Code의 최신 업데이트에서 MCP(Model Context Protocol) 서버를 로컬에서 바로 붙일 수 있게 됐습니다.
설정 파일 하나만 추가하면 3분 안에 연결 완료.

## 3. 바로 써먹기
- [ ] .claude/mcp.json 파일을 프로젝트 루트에 생성
- [ ] mcpServers 키에 서버 이름과 command/args 지정
- [ ] Claude Code 재시작 후 /mcp 명령으로 연결 확인

## 4. 예시
{
  "mcpServers": {
    "my-tools": {
      "command": "node",
      "args": ["./mcp-server.js"]
    }
  }
}

## 5. 추가 팁
- stdio 전송 방식 외에 SSE도 지원 예정
- 디버깅 시 MCP_DEBUG=1 환경변수 활용
- 여러 서버를 동시에 연결하면 도구가 합쳐져 사용 가능
